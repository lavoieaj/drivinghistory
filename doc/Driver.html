    class Driver

    This class defines Driver objects within the DrivingHistory database

    Each Driver will typically reside among other drivers within an array belonging to a parent DrivingHistory object. Driver names are unique. A Driver may have several trips.

    Attributes

    name[R]

    The first name of the driver (e.g. “Alex”)

    trips[R]

    An array of Trips belonging to the Driver

def report
  report_string = @name + ": " + miles_driven.to_s + " miles"
  if miles_driven > 0
    report_string += " @ " + avg_speed.to_s + " mph"
  end
  return report_string
end
